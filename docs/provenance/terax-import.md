# Terax Import Provenance

Millrace Editor is derived from Terax, an Apache-2.0 licensed open-source
AI-native terminal and editor project.

## Upstream

- Project: Terax
- Repository: https://github.com/crynta/terax-ai
- License: Apache-2.0
- Imported from local reference checkout: `ref/cloned-repos/terax-ai/`

## Import Rules

- The reference checkout remains read-only input.
- Product implementation lives in `dev/editor/millrace-editor/`.
- Planning material remains in `lab/`.
- Terax license terms and attribution remain preserved.

## Initial Product Direction

Millrace Editor keeps the Terax desktop workbench foundation and adds
Millrace-native work items, plans, queues, traces, approvals, loop packs,
repair actions, and evidence surfaces.
